# Database Documentation

This folder documents the StayFlow AI PostgreSQL data model and database operations.

Use it for:

- Entity relationship notes and diagrams.
- Table, index, and constraint conventions.
- Migration history and operational guidance.
- Seed data documentation.
- Multi-company isolation rules.
- Data retention, soft delete, and audit field conventions.

Database documentation should complement Entity Framework Core migrations and model configurations without duplicating every implementation detail.
