# Prompt Documentation

This folder captures AI prompts and prompt engineering notes used while building StayFlow AI.

Use it for:

- Reusable prompts for development, review, testing, and documentation.
- Prompt iterations that affect product behavior.
- AI assistant instructions for domain-specific workflows.
- Notes on evaluation criteria and expected output formats.
- Prompt templates for future automation or concierge features.

Prompt documentation should avoid secrets, credentials, private customer data, and any sensitive operational details.
