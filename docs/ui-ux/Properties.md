# Properties

## Purpose

The properties experience helps hosts and property managers manage rental profiles and the knowledge used by the AI concierge.

## Primary Content

- Property name and address.
- Amenities.
- House rules.
- Emergency contacts.
- Local recommendations.
- Knowledge articles.
- Active or inactive status.

## UX Goals

- Make property setup feel guided but efficient.
- Group guest-facing information into clear sections.
- Support quick edits to frequently changing content.
- Show whether a property is ready for AI concierge use.

## States

- Property list.
- Property detail.
- Create property.
- Edit property.
- Empty nested content sections.
- Validation errors.
- Soft-deleted or inactive property.

## Future Considerations

Add readiness scoring, AI knowledge quality checks, and import tools once property onboarding patterns are validated.
