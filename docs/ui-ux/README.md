# UI/UX Documentation

This folder documents the user experience and interface design direction for StayFlow AI.

Use it for:

- Product design principles and interaction patterns.
- User flows for hosts, property managers, guests, and internal operators.
- Wireframe notes, usability findings, and design review outcomes.
- Accessibility, localization, and responsive design guidance.
- Copywriting, tone, and interface content standards.

UI/UX documentation should help product, design, and engineering teams build a consistent experience across web, mobile, and WhatsApp touchpoints.
