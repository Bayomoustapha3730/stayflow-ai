# API Documentation

This folder documents StayFlow AI API contracts and integration behavior.

Use it for:

- Endpoint summaries and route groups.
- Request and response schemas.
- Standardized `ApiResponse<T>` examples.
- Pagination, validation, and error response conventions.
- Authentication and authorization requirements once enabled.
- External webhook contracts and callback payloads.

API documentation should stay aligned with Swagger/OpenAPI output and include practical examples for frontend and integration developers.
