# Architecture Documentation

This folder documents the technical architecture of StayFlow AI.

Use it for:

- System overview diagrams and service boundaries.
- Backend architecture and Clean Architecture conventions.
- Integration patterns for WhatsApp, AI providers, payments, email, and external services.
- Cross-cutting concerns such as authentication, authorization, audit logging, observability, and error handling.
- Deployment and infrastructure architecture as the platform matures.

Architecture documentation should explain why the system is shaped the way it is and link to relevant code, decisions, and database documentation.
