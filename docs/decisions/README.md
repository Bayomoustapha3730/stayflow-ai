# Decision Records

This folder stores important product and engineering decisions for StayFlow AI.

Use it for:

- Architecture Decision Records (ADRs).
- Technology selection notes.
- Data modeling tradeoffs.
- Security and compliance decisions.
- Changes to platform direction or module boundaries.
