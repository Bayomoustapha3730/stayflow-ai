# Security Documentation

This folder documents security practices, controls, and risks for StayFlow AI.

Use it for:

- Authentication, authorization, and permission model notes.
- Tenant and company isolation requirements.
- Threat modeling and risk assessments.
- Secrets management and environment configuration guidance.
- Data protection, audit logging, and incident response procedures.

Security documentation should be kept current as the platform handles more guest, host, payment, and operational data.
