# Mission

## Purpose

This document defines the near-term mission that guides product delivery for StayFlow AI.

## Mission Statement

StayFlow AI helps Airbnb hosts and property managers deliver reliable guest support on WhatsApp by combining structured property knowledge, automation, and AI-assisted concierge workflows.

## Near-Term Focus

- Build a strong backend foundation for companies, properties, users, guests, and conversations.
- Provide property managers with clear tools for managing property knowledge.
- Support WhatsApp-first guest interaction workflows.
- Establish secure authentication, authorization, audit logging, and tenant isolation.
- Create documentation and engineering practices that support repeatable delivery.

## Success Criteria

- Hosts can onboard properties and guest-facing knowledge quickly.
- Guests receive timely and accurate answers.
- Operators can trace actions, errors, and escalations.
- Product changes remain aligned with security and business goals.

## Non-Goals

The mission does not require replacing human property managers. StayFlow AI should automate repetitive work and escalate high-risk or uncertain cases.
