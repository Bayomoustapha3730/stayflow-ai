# Recommendations

## Purpose

Recommendations help guests discover useful nearby places and services during their stay.

## Examples

- Restaurants.
- Cafes.
- Grocery stores.
- Pharmacies.
- Transport options.
- Attractions.
- Laundry services.
- Medical clinics.

## Product Requirements

- Recommendations should be property-specific.
- Each recommendation should include a name and category.
- Optional fields should support descriptions, addresses, and phone numbers.
- Recommendations should be available to AI concierge workflows.
- Inactive or deleted recommendations should not be included in guest-facing answers.

## Content Guidance

Recommendations should be practical and local. Descriptions should include why the place is useful, how far it is from the property when known, and any important access details.

## Future Enhancements

- Distance from property.
- Map links.
- Opening hours.
- Host favorites.
- Guest feedback on recommendations.
