# House Rules

## Purpose

House rules define guest expectations and property restrictions. They help prevent misunderstandings and support consistent guest communication.

## Examples

- No smoking.
- Quiet hours.
- Check-in and check-out times.
- Visitor policy.
- Pet policy.
- Party restrictions.
- Waste disposal instructions.
- Security procedures.

## Product Requirements

- Hosts should be able to add multiple house rules per property.
- Each rule should include a title and clear description.
- House rules should be visible in property details.
- AI concierge workflows should use house rules when answering guest questions.
- Inactive or deleted rules should not be included in guest-facing responses.

## Content Guidance

Rules should be written in guest-friendly language and include consequences or escalation paths only when necessary. Avoid vague rules that require interpretation.
